How it works

01
Create Account
Sign in with X/Twitter. Connect your LinkedIn. Your credentials stay yours.
02
Add API Keys
Bring your own Anthropic & Perplexity keys. We never store or proxy them.
03
Generate API Key
Create your personal API key from the dashboard. This authenticates MCP requests.
04
Plug & Play
Add the MCP config to Claude Code or Claude Desktop. Start posting from any AI workflow.

MCP Server Configuration

Add this to your Claude Code or Claude Desktop config

JSON 
{
  "mcpServers": {
    "social-agents": {
      "command": "python",
      "args": ["/path/to/Social-agents/mcp_server.py"],
      "env": {
        "SOCIAL_AGENTS_URL": "https://twitter.dagent.shop",
        "SOCIAL_AGENTS_API_KEY": "your-api-key-from-dashboard"
      }
    }
  }
}

Install: pip install mcp httpx — Download mcp_server.py from the repo
